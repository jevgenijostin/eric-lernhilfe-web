# eric-lernhilfe-web

Web-hosted (GitHub Pages) versions of Eric's learning apps, so they open as normal
websites on iPhone/iPad (Telegram can't open raw `.html` attachments on iOS).

- **Schach-Trainer** (`index.html`) — German chess trainer: play vs the computer with
  legal-move hints and kid-friendly explanations. Source lives in
  `eric-lernhilfe/schach-trainer/` (built with `python build.py`).

Add to the iPhone home screen via Safari → Share → "Zum Home-Bildschirm" for an app-like icon.
